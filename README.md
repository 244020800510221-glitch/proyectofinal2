# CECYTECA

Sistema de gestión de biblioteca escolar para CeCyTe Baja California.

Slogan: **Cada página impulsa tu futuro en CECYTECA**

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript estricto
- Tailwind CSS
- Zustand + persist middleware
- React Hook Form
- Lucide React

## Instalación

```bash
npm install
npm run dev
```

Abre en el navegador: **http://localhost:3050** (puerto fijo para CECYTECA). La ruta de login es **http://localhost:3050/login**.

Si antes veías “puerto 3000 en uso” y Next pasaba a 3001, era normal: otro programa ocupaba el 3000 y por eso la app “no cargaba” si abrías la URL equivocada.
