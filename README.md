# CECYTECA

Sistema de gestión de biblioteca escolar para CeCyTe Baja California.

Slogan: **Cada página impulsa tu futuro en CECYTECA**

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript estricto
- Tailwind CSS
- Zustand + persist middleware
- React Hook Form
- Lucide React

## Instalación

```bash
npm install
npm run dev
```
